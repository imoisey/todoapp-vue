# TodoApp

Приложение для списка задач на Vue + Symfony 5

## Installation

Для запуска приложения требуется: 

* docker `>=18.06`
* docker-compose `>=1.24`

Сборка приложения:

```bash
make init
```

Запуск приложения:

```bash
make up
```

## Usage

* [http://localhost:8080](http://localhost:8080) - frontend
* [http://localhost:8081](http://localhost:8081) - api
